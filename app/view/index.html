Classify 102 Flowers

This model was trained by Gavin Francis, using the fast.ai library, based on the Oxford 102 Flowers data set
created by Maria-Elena Nilsback and Andrew Zisserman,
Visual Geometry Group, Department of Engineering Science
University of Oxford, United Kingdom
The model achieved an accuracy of 80% on a test set of 102 flowers
It works best on full frame close up images of a single flower
